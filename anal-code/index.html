Часто задаваемые вопросы

Что такое CodeFusion?

CodeFusion v3.0 — это AI-анализатор исходного кода с поддержкой 50+ языков. Включает загрузку с GitHub, diff-сравнение, security scan, code metrics, command palette и minimap.

Как загрузить код с GitHub?

Вставьте URL репозитория или файла во вкладке «GitHub». CodeFusion автоматически распарсит URL и загрузит файлы через GitHub API.

Безопасно ли загружать файлы?

Да. Все файлы обрабатываются локально в браузере через File API. Данные не отправляются на сервер. AI-анализ также работает локально.

Какие форматы поддерживаются?

50+ форматов: HTML, CSS, JS, TS, Python, Java, PHP, Go, Rust, C/C++, C#, Ruby, Swift, Kotlin, SQL, XML, JSON, YAML, Markdown и другие.

Нужна ли регистрация?

Нет. CodeFusion полностью бесплатен и работает без регистрации. Все операции происходят локально в браузере.
